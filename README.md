## parcel plugin static files copy
- https://www.npmjs.com/package/parcel-plugin-static-files-copy
- `npm i -D parcel-plugin-static-files-copy`
- `package.json` 파일로 이동 하여 추가

```json
"staticFiles": {
    "staticPath": "static"
  }

```
